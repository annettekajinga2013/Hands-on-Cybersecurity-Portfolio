## 📝 Study Plan by Domain

### 1️⃣ Cloud Security (GCP, AWS, Azure)
- IAM & role-based access control  
- VPC / Networking & firewall configuration  
- Cloud storage & encryption practices  
- Monitoring, logging, and alerting (Cloud Operations, CloudTrail, Azure Sentinel)  
- Cloud incident response & threat detection  
- Hands-on labs: Google ACE Labs, AWS Security Labs, Azure Security Labs  

**Time Estimate:** 3–4 months for PCE readiness  

---

### 2️⃣ Networking & Systems (Cisco CCNA)
- OSI model & TCP/IP fundamentals  
- VLANs, routing, switching, subnetting  
- ACLs & network security best practices  
- Network troubleshooting & packet analysis  
- Hands-on labs: Cisco Packet Tracer, lab routers & switches  

**Time Estimate:** 6–7 months  

---

### 3️⃣ SOC & Blue Team / Defensive Security
- SIEM fundamentals (Splunk, Elastic, ELK)  
- Log analysis (Windows Event Logs, Sysmon, Linux logs)  
- Threat hunting & MITRE ATT&CK techniques  
- Incident response lifecycle & automation (SOAR, Python/PowerShell)  
- Cloud SOC monitoring & detection  

**Time Estimate:** Ongoing alongside hands-on labs (1–3 months per lab series)  

---

### 4️⃣ Offensive Security & Pentesting
- Reconnaissance, enumeration, and vulnerability assessment  
- Exploitation techniques (web, network, privilege escalation)  
- Reporting & documentation standards  
- Hands-on labs: Hackviser CAPT, TryHackMe, Hack The Box  

**Time Estimate:** 2–3 months for CAPT readiness  

---

### 5️⃣ Management-Level / Leadership Certifications (ISC²)
- CISSP domains: Security & Risk Management, Asset Security, Security Architecture & Engineering, Communication & Network Security, Identity & Access Management, Security Assessment & Testing, Security Operations, Software Development Security  
- CCSP domains: Cloud Concepts, Architecture, Governance, Compliance, Cloud Security Operations, Legal & Risk  

**Time Estimate:** 10–15 months (after foundational certs & hands-on experience)
